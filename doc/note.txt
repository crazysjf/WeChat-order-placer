# 次品处理流程

## 次品登记：
方法一：
进入库存 => 快销退货，扫码进行次品登记。开启检验供应商不要勾选。
该方法会校验唯一码的唯一性，不会重复扫描，缺点是确认之后，退货单不能修改。

方法二：
进入库存 => 采购退货 => 创建新单据 => 创建退货单
仓库务必选销退仓，供应商随便选。

创建好后，在下面勾选“使用逐渐扫描录入商品”。
此处不会校验唯一码，要注意不要重复扫描。
完成后无需确认生效，直接导出即可。



## 次品登
文件名不要做任何修改，保存到当日报单文件夹。

## 其他准备记表导出：
进入库存 => 采购退货，可以看到刚才登记的次品的单据。忽略供应商。
确认日期时间和内容后，勾选，点击导出勾选单据。
普通商品资料要随时导出一份最新版本，放入报单文件夹最上层供引用。

## 第二天次品插入报单报表及报单流程
第二天报单时，到第一天文件所在文件夹里面寻找次品登记报表。
有的话，逐级向上寻找商品资料报表。

计算出次品的供应商和供应商款号，插入供应商正常报单下面。
格式例：
11186-美雪， 8819次品，颜色尺码无，换3
